Module registration

Registering a module is the first thing your module needs to do. This registers your module with the core phpSecureSite system, and gives access to functionality for storing and retreiving data and configuration options, among other things.

The module is registered through a call to phpss_module_register(). This call should usually be placed at the very top of the script. It takes It takes the name of the module as the first argument. and the optional second argument lets you specify the type of module you are registering. It defaults to a normal module.
