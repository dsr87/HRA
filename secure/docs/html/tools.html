Chapter 7. Tools

phpSecureSite includes tools for handling various administrative tasks. These can be found in the tools/ directory.

dbclean (database cleanups)

tools/dbclean.php

The dbclean tool performs cleanups of the phpSecureSite database, such as removing outdated and corrupt data. It can also remove old and unused session to make the system faster, but this will remove session data which may be useful when reading logs etc.

To keep the database relatively small and the overall performance of the system good, it is recommended that you run this script in a cron-job once every night. It can also be run via your browser by putting it under the document root of your server.

Configuration is done in the tool file itself, near the top.

Configuration options

phpss_path

The full path to the phpss.php file. dbclean needs to load phpSecureSite to fetch its config, access the database backend and take advantage of the infrastructure it provides.

idlelimit

The number of idle hours before a session is considered inactive. Data which is no longer useful will be removed for these sessions.

removesessions

When true, all session data for inactive sessions is removed. This should improve the performance of the overall system, but will remove data which may be useful when reading logs or for historical reference.
